# FitHire by Coach360

**Purpose:** Connect verified fitness professionals with relevant job opportunities and help clubs/studios identify aligned candidates using a deterministic FitScore ranking system.

---

## Description

FitHire is a platform designed to streamline the hiring process for fitness professionals and organizations. Coaches create verified profiles with certifications, experience, and availability. Clubs and studios post job listings with specific requirements and culture indicators. FitHire calculates a **FitScore** to rank matches between coaches and jobs, helping both sides find optimal alignment quickly.

**Key Features:**
- Deterministic FitScore ranking based on certifications, experience, availability, culture, and engagement signals
- Top-N matching for coaches and jobs
- Admin-verified tags for movement, instruction style, and lifestyle
- Multi-level access for brands, regions, and locations
- Threshold-based display for relevant matches

**Differentiators:**
- Transparent, non-black-box scoring engine
- Manual verification ensures high data integrity
- Designed to scale for multi-location organizations
